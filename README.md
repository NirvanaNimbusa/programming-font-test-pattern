# programming-font-test-pattern
This test pattern can be used to check if a programming font's letters are clearly distiguishable.

    o0O s5S z2Z !|l1Iij {([|})] .,;: ``''"" 
    a@#* vVuUwW <>;^°=-~ öÖüÜäÄßµ \/\/ 
    the quick brown fox jumps over the lazy dog
    THE QUICK BROWN FOX JUMPS OVER THE LAZY DOG
    0123456789 &-+@ for (int i=0; i<j; ++i) { }
